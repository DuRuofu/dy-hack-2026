# Demo 1：视频链接解析 + 关键帧提取

> 这是一切的基础能力，两个赛道都绕不开。

## 目标

输入一个视频链接（抖音/B站/YouTube），自动解析并提取：
- 视频标题、描述、作者等元信息
- 关键帧图片（按场景变化或固定间隔抽帧）
- 音频文件（用于后续转文字）
- 字幕（如有）

## 技术栈

| 组件 | 方案 | 理由 |
|------|------|------|
| 链接解析 | `yt-dlp-exec` | yt-dlp 的 Node.js 封装，支持抖音/B站/YouTube |
| 视频处理 | `fluent-ffmpeg` | Node.js 下最成熟的 FFmpeg 封装 |
| 音频提取 | `fluent-ffmpeg` | 从视频中分离音频，输出 mp3/wav |
| 字幕提取 | `yt-dlp-exec --write-subs` | 如果视频有字幕，直接下载 |

## 核心流程

```
视频链接
  │
  ├─→ yt-dlp 获取元信息（标题、描述、时长、封面）
  ├─→ yt-dlp 下载视频文件
  ├─→ ffmpeg 按间隔抽帧（每 N 秒一帧）或按场景检测抽帧
  ├─→ ffmpeg 提取音频
  └─→ 返回 { meta, frames[], audio_path }
```

## 关键代码示例

```typescript
import ytdl from 'yt-dlp-exec'
import ffmpeg from 'fluent-ffmpeg'
import fs from 'fs/promises'
import path from 'path'

interface VideoMeta {
  title: string
  description: string
  duration: number
  uploader: string
}

async function parseVideo(url: string): Promise<VideoMeta> {
  // 获取元信息
  const info = await ytdl(url, { dumpJson: true }) as unknown as VideoMeta

  // 下载视频
  await ytdl(url, { output: 'output.mp4' })

  // 每 5 秒抽一帧
  await new Promise<void>((resolve, reject) => {
    ffmpeg('output.mp4')
      .outputOptions('-vf', 'fps=1/5')
      .output('frames/frame_%04d.jpg')
      .on('end', () => resolve())
      .on('error', reject)
      .run()
  })

  // 提取音频
  await new Promise<void>((resolve, reject) => {
    ffmpeg('output.mp4')
      .noVideo()
      .audioCodec('libmp3lame')
      .save('audio.mp3')
      .on('end', () => resolve())
      .on('error', reject)
      .run()
  })

  return info
}
```

## 依赖

```bash
pnpm add yt-dlp-exec fluent-ffmpeg
pnpm add -D @types/fluent-ffmpeg
```

系统需安装 `ffmpeg`（`brew install ffmpeg`）和 `yt-dlp`（`brew install yt-dlp`）。

## 输出格式

```json
{
  "title": "视频标题",
  "description": "视频描述",
  "duration": 120,
  "uploader": "作者名",
  "frames": ["frames/frame_0001.jpg", "frames/frame_0002.jpg", ...],
  "audio": "audio.mp3",
  "subtitles": "subs.srt"
}
```

## 注意事项

- 抖音链接解析可能需要 cookie，比赛时提前测试
- 抽帧频率可调：场景变化多的视频可以按场景检测（`ffmpeg -vf "select='gt(scene,0.3)'"`）
- 下载速度取决于网络，比赛现场提前确认网速
