# Demo 3：视频内容结构化摘要

> 赛道三"内容理解→重组"的核心能力。

## 目标

输入一组关键帧 + 音频转录文本，输出：
- 视频整体摘要
- 时间线分段（每个时间段讲了什么）
- 关键信息点提取
- 话题标签

## 技术栈

| 组件 | 方案 | 理由 |
|------|------|------|
| 音频转文字 | OpenAI Whisper API（云端） | 精度高，不依赖本地算力，Node.js 直接调用 |
| 备选转文字 | 通义听悟 API | 国内访问快，中文效果好 |
| 关键帧理解 | 复用 Demo 2 | 已有的图片理解能力 |
| 摘要生成 | `@anthropic-ai/sdk` | 长文本理解能力强，适合结构化摘要 |

## 核心流程

```
Demo 1 的输出
  │
  ├─→ audio.mp3 → Whisper API → 转录文本（带时间戳）
  ├─→ frames[] → Demo 2 → 每帧场景描述
  │
  └─→ 合并上下文 → Claude API → 结构化摘要
```

## 关键代码示例

```typescript
import Anthropic from '@anthropic-ai/sdk'
import OpenAI from 'openai'
import fs from 'fs/promises'

interface TranscriptSegment {
  start: number
  end: number
  text: string
}

interface VideoSummary {
  summary: string
  segments: { time: string; topic: string; key_points: string[] }[]
  tags: string[]
}

async function transcribeAudio(audioPath: string): Promise<TranscriptSegment[]> {
  const openai = new OpenAI()
  const file = await fs.readFile(audioPath)

  const response = await openai.audio.transcriptions.create({
    file: new File([file], 'audio.mp3', { type: 'audio/mpeg' }),
    model: 'whisper-1',
    response_format: 'verbose_json',
  })

  return (response as any).segments.map((seg: any) => ({
    start: seg.start,
    end: seg.end,
    text: seg.text,
  }))
}

async function summarizeVideo(
  segments: TranscriptSegment[],
  frameDescriptions: string[],
): Promise<VideoSummary> {
  let context = '=== 音频转录 ===\n'
  for (const seg of segments) {
    context += `[${Math.round(seg.start)}s-${Math.round(seg.end)}s] ${seg.text}\n`
  }

  context += '\n=== 关键帧描述 ===\n'
  frameDescriptions.forEach((desc, i) => {
    context += `帧${i + 1}: ${desc}\n`
  })

  const client = new Anthropic()
  const response = await client.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 2048,
    messages: [{
      role: 'user',
      content: `请基于以下视频内容，返回结构化 JSON：

${context}

返回格式：
{
  "summary": "一句话总结",
  "segments": [
    {"time": "0-30s", "topic": "话题", "key_points": ["要点1"]}
  ],
  "tags": ["标签1", "标签2"]
}`,
    }],
  })

  return JSON.parse(response.content[0].text)
}
```

## 优化方向

- **分段粒度可调**：30秒/60秒/按话题切换自动分段
- **多语言支持**：Whisper API 支持中英文混合转录
- **摘要长度控制**：比赛展示时可能需要精简版
