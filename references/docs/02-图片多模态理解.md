# Demo 2：单张图片多模态理解

> 赛道一的核心能力——"认出画面"并理解意图。

## 目标

输入一张图片，输出结构化的理解结果：
- 场景描述（这是什么地方/什么场景）
- 物体识别（画面中有什么东西）
- OCR 文字提取
- 情绪/意图分析
- 结构化 JSON 输出

## 技术栈

| 组件 | 方案 | 理由 |
|------|------|------|
| 多模态理解 | `@anthropic-ai/sdk` | Claude Sonnet，一次调用搞定所有识别任务 |
| 备选方案 | 通义千问 VL API | 国内访问快，免费额度多 |
| 图片预处理 | `sharp` | Node.js 高性能图片处理库，压缩/裁剪 |

## 核心流程

```
图片文件
  │
  ├─→ sharp 预处理（压缩到合适尺寸，控制 token）
  ├─→ 调用多模态 API（base64 编码图片 + prompt）
  └─→ 返回结构化 JSON
```

## 关键代码示例

```typescript
import Anthropic from '@anthropic-ai/sdk'
import sharp from 'sharp'
import fs from 'fs/promises'

interface ImageAnalysis {
  scene: string
  objects: string[]
  text_ocr: string
  mood: string
  tags: string[]
}

async function understandImage(imagePath: string): Promise<ImageAnalysis> {
  // 预处理：压缩图片控制 token 消耗
  const resized = await sharp(imagePath)
    .resize(1568, 1568, { fit: 'inside' })
    .jpeg({ quality: 80 })
    .toBuffer()

  const imgB64 = resized.toString('base64')

  const client = new Anthropic()
  const response = await client.messages.create({
    model: 'claude-sonnet-4-20250514',
    max_tokens: 1024,
    messages: [{
      role: 'user',
      content: [
        {
          type: 'image',
          source: {
            type: 'base64',
            media_type: 'image/jpeg',
            data: imgB64,
          },
        },
        {
          type: 'text',
          text: `请分析这张图片，返回 JSON 格式：
{
  "scene": "场景描述",
  "objects": ["物体1", "物体2"],
  "text_ocr": "图片中的文字",
  "mood": "情绪/氛围",
  "tags": ["标签1", "标签2"]
}`,
        },
      ],
    }],
  })

  return JSON.parse(response.content[0].text)
}
```

## 优化方向

- **Prompt 调优**：根据比赛关键词定制识别维度（美食→识别菜品/食材；穿搭→识别品牌/风格）
- **批量处理**：对一组图片（如视频关键帧）做批量理解
- **Token 控制**：图片压缩到 1568px 以内，JPEG 质量 80%，节省 API 费用
