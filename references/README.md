# 抖音黑客松 - AI Demo 预研仓库

> 8 小时比赛，提前备好零件，当天只做组装。

## 背景

参加抖音举办的黑客松比赛，涉及两条赛道：

- **赛道一：视觉搜索** — 图像理解 + 场景延伸 + 内容创造
- **赛道三：抖音精选社区** — 视频内容理解 → 重组 → 应用闭环

比赛当天会有关键词进一步限定方向，因此需要准备**通用、可组合的 Demo 模块**，当天根据关键词快速二次创作。

## 设计原则

- **轻量优先**：Node.js/TS + 现成 API，不用重型框架
- **快速出活**：每个 Demo 独立可运行，5 分钟内跑通
- **模块可组合**：Demo 之间可以串联，像积木一样拼装
- **UI 极简**：Vue 3 + Vite 做简单界面，比赛展示够用即可

## 技术栈

| 层级 | 选型 |
|------|------|
| 语言 | Node.js / TypeScript |
| 包管理 | pnpm |
| 前端 | Vue 3 + Vite |
| 后端 | Hono |

## 运行 Demo

```bash
# 进入某个 Demo 目录
cd demos/demo1-video-parse

# 安装依赖
pnpm install

# 启动后端
pnpm dev:server

# 新开终端，启动前端
pnpm dev:client

# 浏览器打开 http://localhost:5173
```
