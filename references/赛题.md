## 🎯 赛道一：视觉搜索

**核心定位：从“认出”到“创造”的视觉交互革新**

- **基础能力**：基于图像识别的成熟场景，如识别物体、找同款、定位地标等。
- **升级方向**：不只是“认出画面”，更要让AI**看懂画面意图、主动理解场景**，甚至能基于图像**创造新内容、延伸服务**。
- **核心问题**：打破“拍下只是识别”的惯性，探索“按下快门之后，还能发生什么？”的创新应用。
- **典型方向**：图像理解驱动的内容生成、场景交互、个性化服务等。

## 🎯 赛道三：抖音精选社区赛题

**核心定位：面向全品类视频内容的AI赋能闭环**

- **覆盖场景**：面向游戏、二次元、科技（含AI）、科普、人文社科、职场、课程、旅行、美食、法律、财经、时政等几乎所有短视频/中长视频领域。
- **核心目标**：让AI从“理解内容”出发，完成**内容理解→重组→应用**的完整创新闭环，最终赋能创作者与用户，带来社区互动的新可能。
- **开放赛题**：所有符合“创新AI应用”方向的原生AI创新均可参赛，优质应用可直接在抖音精选App上分发落地。

### 两者的关键区别

| 维度 | 赛道一：视觉搜索 | 赛道三：抖音精选社区赛题 |
| :--- | :--- | :--- |
| **核心载体** | 图像/视觉内容 | 全品类视频内容（短/中长视频） |
| **核心能力** | 图像理解与场景延伸 | 内容理解、重组与应用闭环 |
| **落地场景** | 通用视觉交互创新 | 抖音社区生态内的内容创作与互动 |
| **开放度** | 聚焦“视觉+AI”的垂直创新 | 全品类内容的原生AI创新，落地路径更直接 |
