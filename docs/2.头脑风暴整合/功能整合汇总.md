# 功能整合汇总

---

## 三份文档核心观点

| 成员 | 核心视角 | 关键词 |
|------|---------|--------|
| DuRuofu | 用户场景 + 开发者模块拆解 + MVP 分级 | 四种衣橱来源、场景驱动推荐、P0/P1/P2 |
| 田 | 产品级需求文档，强调双库 + AI助手 + 推荐比例 | 视频提取溯源、悬浮AI助手、60/30/10推荐 |

## 共识功能（三人/两人均提到）

| 功能 | 提出者 | 说明 |
|------|--------|------|
| 抖音视频穿搭提取 | DuRuofu + 田 | 输入视频链接，AI 提取服装单品 |
| 用户个人衣橱 | DuRuofu + 田 | 上传/管理自己的衣物 |
| AI 穿搭推荐 | DuRuofu + 田 | 根据场景/偏好推荐搭配方案 |
| 多场景支持 | DuRuofu + 田 | 通勤、约会、休闲、运动等 |
| 单品信息识别 | DuRuofu + 田 | 识别颜色、品类、风格、季节 |

## 差异功能（仅一人提出）

| 功能 | 提出者 | 说明 |
|------|--------|------|
| 淘宝购物数据同步 | DuRuofu | 读取淘宝购买记录，自动导入衣物 |
| 拍照上传识别 | DuRuofu | 对着衣柜拍照，AI 自动识别分类 |
| 缺件提示 | DuRuofu | 推荐方案中提示衣橱缺少的单品 |
| 一键复刻博主穿搭 | DuRuofu | 用自己衣橱的相似单品替换博主同款 |
| 穿搭历史/月度回顾 | DuRuofu | 记录每日穿搭，生成回顾 |
| 衣橱筛选/标记 | DuRuofu | 按品类/颜色/季节筛选，标记常穿/淘汰 |
| 视频穿搭公共库 | 田 | 所有用户共享的穿搭素材库 |
| 单品溯源（找同款） | 田 | 匹配展示同款商品来源 |
| 悬浮 AI 穿搭助手 | 田 | 页面悬浮球，随时唤起交互 |
| 用户身体档案 | 田 | 录入身高/体重/年龄/性别/职业 |
| 60/30/10 推荐比例 | 田 | 60% 适配 + 30% 差异 + 10% 跨风格 |
| 全品类覆盖（包包/配饰） | 田 | 推荐不只衣服，含鞋/包/配饰 |
