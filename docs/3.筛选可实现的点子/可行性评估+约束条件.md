# 可行性评估

> 黑客松 8 小时内能做完吗？逐个过一遍。
> 筛选标准：视觉观感好 + 抖音黑客松演示效果强 + 8 小时内可交付。
> **核心约束：视频穿搭提取不可取消。**

## 约束条件

1. **前后端分离** — 前端和后端独立开发、独立部署，通过 API 通信
2. **Web 页面，响应式适配** — 前端使用 Web 技术（HTML/CSS/JS），同时适配手机端和电脑端浏览器
3. **前端框架 Vue 3** — 使用 Vue 3 + Vite 构建前端

## 评估维度

- **技术难度**：低 / 中 / 高
- **预估工时**：X 小时
- **依赖**：需要什么前置条件
- **Demo 复用**：能否直接用 references 里的模块
- **视觉观感**：演示时观众能否一眼看懂
