# 环境变量配置

> 所有敏感配置和可变参数统一通过 `.env` 文件管理，不提交到 Git。

## 后端 `.env`

```bash
# ===== AI 模型切换 =====
AI_PROVIDER=alibaba_bailian    # alibaba_bailian | volcano | minimax

# 阿里百炼
ALIBABA_BAILIAN_API_KEY=sk-xxx

# 火山大模型（字节跳动）
VOLCANO_API_KEY=xxx
VOLCANO_ENDPOINT=https://ark.cn-beijing.volces.com/api/v3

# MiniMax 2.7
MINIMAX_API_KEY=xxx
MINIMAX_GROUP_ID=xxx

# ===== 数据库 =====
DATABASE_URL=postgresql://user:pass@host:5432/dazi

# ===== 阿里云 OSS =====
OSS_REGION=oss-cn-hangzhou
OSS_ACCESS_KEY_ID=xxx
OSS_ACCESS_KEY_SECRET=xxx
OSS_BUCKET=dazi-images

# ===== 服务端口 =====
PORT=3000
```

## 前端 `.env`

```bash
# 后端 API 地址
VITE_API_BASE_URL=http://localhost:3000/api
```

## `.env.example`

项目根目录提供 `.env.example` 作为模板，新成员 `cp .env.example .env` 后填入自己的密钥即可。
