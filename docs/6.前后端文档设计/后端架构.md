# 后端架构

## 技术选型

- Node.js + TypeScript
- Hono 框架
- SQLite + Drizzle ORM
- 通义千问 (DashScope) 多模态 API

## 目录结构

```
backend/src/
├── routes/         # 路由
│   ├── wardrobe.ts
│   ├── outfit.ts
│   ├── recognize.ts
│   └── inspiration.ts
├── services/       # 业务逻辑
│   ├── ai.ts       # AI 调用封装
│   └── taobao.ts   # 淘宝数据
├── db/             # 数据库
│   ├── schema.ts   # 表结构
│   └── index.ts    # 连接
├── utils/          # 工具函数
└── index.ts        # 入口
```

## API 清单

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/recognize | 图片识别衣物 |
| GET | /api/wardrobe | 获取衣橱列表 |
| POST | /api/wardrobe | 添加衣物 |
| DELETE | /api/wardrobe/:id | 删除衣物 |
| POST | /api/outfit/recommend | 获取搭配推荐 |
| GET | /api/inspiration | 获取灵感穿搭 |

## 数据库表

| 表 | 说明 |
|----|------|
| clothes | 衣物（品类、颜色、风格、图片URL） |
| outfits | 搭配方案记录 |
| users | 用户信息和偏好 |
