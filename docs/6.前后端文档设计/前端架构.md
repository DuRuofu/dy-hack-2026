# 前端架构

## 技术选型

- Vue 3 + Composition API + `<script setup>`
- Vite 构建
- Tailwind CSS 样式
- Vue Router 路由
- Pinia 状态管理

## 目录结构

```
frontend/src/
├── views/          # 页面组件
│   ├── Home.vue        # 首页：场景选择 + 推荐
│   ├── Upload.vue      # 拍照上传识别
│   ├── Video.vue       # 抖音视频解析
│   ├── Taobao.vue      # 淘宝链接导入
│   ├── Wardrobe.vue    # 我的衣橱（只读网格）
│   ├── Outfit.vue      # 搭配推荐详情
│   └── MixMatch.vue    # 自由搭配 + AI 评价
├── components/     # 通用组件
│   ├── ClothingCard.vue   # 衣物卡片
│   ├── OutfitCard.vue     # 搭配方案卡片
│   ├── SceneCard.vue      # 场景卡片
│   ├── ScoreBadge.vue     # AI 评分徽章
│   └── TabBar.vue         # 底部导航
├── stores/         # Pinia 状态
│   ├── wardrobe.ts
│   └── outfit.ts
├── api/            # Axios 接口封装
├── router/         # Vue Router 配置
├── assets/         # 静态资源
└── App.vue
```

## 状态设计

| Store | 数据 |
|-------|------|
| wardrobeStore | 衣物列表、筛选条件、来源标签 |
| outfitStore | 当前推荐方案、AI 评价结果 |

> 无 userStore — 项目无用户体系，全员共享数据。
