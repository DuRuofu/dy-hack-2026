# 前端架构

## 技术选型

- Vue 3 + Composition API + `<script setup>`
- Vite 构建
- Tailwind CSS 样式
- Vue Router 路由
- Pinia 状态管理

## 目录结构

```
frontend/src/
├── views/          # 页面组件
│   ├── Home.vue
│   ├── Upload.vue
│   ├── Wardrobe.vue
│   ├── Outfit.vue
│   ├── Inspiration.vue
│   └── Profile.vue
├── components/     # 通用组件
├── stores/         # Pinia 状态
├── api/            # 接口封装
├── router/         # 路由配置
├── assets/         # 静态资源
└── App.vue
```

## 状态设计

| Store | 数据 |
|-------|------|
| wardrobeStore | 衣物列表、筛选条件 |
| outfitStore | 当前推荐方案 |
| userStore | 用户信息、偏好 |
