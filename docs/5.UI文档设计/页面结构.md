# 页面结构

> 搭子 App 的页面路由和结构

## 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | / | 场景选择 + 今日推荐 |
| 拍照上传 | /upload | 拍照/选图 + 识别结果 |
| 我的衣橱 | /wardrobe | 衣物列表 + 筛选 |
| 搭配推荐 | /outfit | 推荐方案详情 |
| 灵感广场 | /inspiration | 抖音视频穿搭流 |
| 个人中心 | /profile | 设置、授权 |

## 导航结构

```
TabBar: 首页 | 衣橱 | 拍照(+) | 灵感 | 我的
```

## 交互流程

1. 首页选场景 → 点"帮我搭" → 跳转搭配推荐
2. 底部拍照按钮 → 拍照上传 → 识别结果 → 加入衣橱
3. 灵感页 → 点击搭配 → 查看详情 → "用我的衣服复刻"
