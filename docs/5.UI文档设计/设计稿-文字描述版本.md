## 二、各屏关键布局（文字稿）

### 首页 `/`

- 顶区：产品标题或简短说明（可选）。
- 主体：**场景卡片列表**（图标 + 场景名 + 一句描述，数据来自 `GET /api/scenes`）。
- 辅助：**「自由搭配」** 按钮/卡片，跳 `/mix-match`。
- 底部：**TabBar**（首页 / 衣橱 / 上传 / 推荐 / 淘宝 / 视频）。

### 衣橱 `/wardrobe`

- 顶区：**筛选**（品类、季节、风格等 Chip 或下拉），变更后仍在本页。
- 主体：**衣物网格**（图 + 品类/颜色等标签）；**无删除按钮**。
- 单卡点击进入 **`/wardrobe/:id`**。
- 可选：**「自由搭配」** 入口 → `/mix-match`。

### 衣物详情 `/wardrobe/:id`

- 大图 + 名称/品类/颜色/风格/季节等；可编辑字段与 **保存**（`PUT`）。
- 返回 → 衣橱或上一页。

### 拍照上传 `/upload`

- **拍照 / 选图** 主按钮；预览区。
- 识别结果表单（可修正）+ **保存到衣橱**。
- 无删除、无账号区。

### 视频解析 `/video`

- 输入框：视频链接；**解析** 按钮。
- 结果：**可勾选** 的识别单品列表；**保存选中到衣橱**。

### 淘宝导入 `/taobao`

- 输入框：商品链接；**解析**。
- 预览主图与识别字段；**确认保存到衣橱**。

### 搭配推荐 `/outfit`

- 顶部可显示当前场景名；**换场景** → `/`。
- 主体：**2～3 套** `OutfitCard`（每套含多单品缩略图、分数、理由摘要）。
- 点击整套卡片 → **`/outfit/:id`**。

### 搭配方案详情 `/outfit/:id`

- 横向或纵向排列本套 **上衣 / 下装 / 鞋** 大图；完整理由与评分。
- 点某一单品 → **`/wardrobe/:id`**。

### 自由搭配 `/mix-match`

- 上半：**衣橱网格**，按规则选取上/下/鞋（可用分栏或步骤提示）。
- **让 AI 点评**：同页下方展示分数、`pros` / `cons` / `suggestion`。
