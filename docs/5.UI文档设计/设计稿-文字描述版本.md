## 二、各屏关键布局（文字稿）

### 首页 `/`

- 顶区：产品标题或简短说明（可选）；**职业定制**入口（Chip 或「我的穿搭身份」）→ 与个人中心或首启选择联动。
- 主体：**场景卡片列表**（衣架/单品简笔图标 + 场景名 + 一句描述，数据来自 `GET /api/scenes`），卡片背景宜用浅天蓝 `#E0F7FF` 与主色描边呼应。
- 辅助：**「自由搭配」** 按钮/卡片，跳 `/mix-match`（图标建议：拼接衣架 + 星星，表示自选与点评）。
- 底部：**TabBar**（首页 / 衣橱 / 上传 / 推荐 / 淘宝 / 视频；若含个人中心则增加「我的」）。

### 衣橱 `/wardrobe`

- 顶区：**筛选**（品类、季节、风格等 Chip 或下拉），变更后仍在本页；筛选条用 `#B0E0E6` 弱分割、选中态用主色描边。
- 主体：**衣物网格**（图 + 品类/颜色等标签）；**无删除按钮**。
- 单卡点击进入 **`/wardrobe/:id`**。
- 可选：**「自由搭配」** 入口 → `/mix-match`。

### 衣物详情 `/wardrobe/:id`

- 大图 + 名称/品类/颜色/风格/季节等；可编辑字段与 **保存**（`PUT`）。
- 返回 → 衣橱或上一页。

### 拍照上传 `/upload`

- **拍照 / 选图** 主按钮；预览区。
- 识别结果表单（可修正）+ **保存到衣橱**。
- 无删除、无账号区。

### 视频解析 `/video`

- 输入框：视频链接；**解析** 按钮。
- 结果：**可勾选** 的识别单品列表；**保存选中到衣橱**。

### 淘宝导入 `/taobao`

- 输入框：商品链接；**解析**。
- 预览主图与识别字段；**确认保存到衣橱**。

### 搭配推荐 `/outfit`

- 顶部可显示当前场景名；**换场景** → `/`。
- 主体：**2～3 套** `OutfitCard`（每套含多单品缩略图、分数、理由摘要）。
- 点击整套卡片 → **`/outfit/:id`**。

### 搭配方案详情 `/outfit/:id`

- 横向或纵向排列本套 **上衣 / 下装 / 鞋** 大图；完整理由与评分。
- 点某一单品 → **`/wardrobe/:id`**。

### 自由搭配 `/mix-match`

- 上半：**衣橱网格**，按规则选取上/下/鞋（可用分栏或步骤提示）。
- **让 AI 点评**：同页下方展示分数、`pros` / `cons` / `suggestion`。

### 个人中心 `/profile`（或与首启职业选择合并）

- **职业身份**：互联网 / 教师 / 公职人员 / 设计师 / 博主 / 摄影师 / 大学生 / 研究生等（分组映射到三大类：职场白领 / 创意自由职业 / 学生）。
- **穿搭偏好**：可选标签（如通勤、校园、约会）仅影响文案与默认场景排序，不改变后端共享数据模型。
- **保存**：写入本地（如 `localStorage`）或 Pinia 持久化，全站按钮样式随职业切换。

## 三、设计规范

整体方向：**天蓝色温柔风**，所有组件围绕**衣架、衣物、鞋履、搭配**意象延展，避免与穿搭无关的装饰主色。

| 元素 | 值 |
|------|---|
| 主色调 | `#87CEEB`（天蓝色） |
| 辅助色 | `#E0F7FF`（浅天蓝）、`#B0E0E6`（淡蓝灰） |
| 强调色 | `#4682B4`（钢蓝色） |
| 字体 | `Inter, "PingFang SC", "Microsoft YaHei", sans-serif` |
| 圆角 | 主按钮 **16px**；次要按钮 **12px**；卡片 **16px** |
| 间距 | 基础单位 **4px**；常用 **8px / 16px / 24px / 32px** |

**补充约定（穿搭主题统一感）**

- 页面背景：大面使用 `#E0F7FF` 或白底 + 顶部/底 `#B0E0E6` 细分隔，主色用于主操作与选中态。
- 图标风格：线性或轻填充，线宽统一；优先衣类符号（衣架、T 恤轮廓、鞋、衣橱格子），职业仅作**小徽标**叠加在角落，不抢主识别。
- 文本色：主文 `#1a1a1a`～`#2d3748`，次级 `#718096`；主按钮上文字用白或近白，浅底按钮用 `#4682B4`。

## 四、职业定制化按钮设计

> 三大类职业共用**同一套天蓝色色相**，仅通过**深浅、边框、图标**区分气质；所有按钮文案与图标须与**穿搭场景**强相关。

**通用尺寸约定**

- 主按钮高度 **48px**（移动端可 **44px**），左右内边距 **24px**；图标区 **20×20px**，与文字间距 **8px**。
- 次按钮高度 **40px**，圆角 **12px**。
- 全宽主按钮在表单页（解析、上传）贴边留白 **16px**。

**交互（全站一致）**

- **Hover**（支持键鼠端）：背景亮度 +4%～6%，或强调色 `#4682B4` 下沿 **1px** 内阴影 `0 2px 8px rgba(70, 130, 180, 0.25)`；图标与文字同步 **translateY(-1px)**。
- **Active / 点击**：缩放 **0.98**，阴影减弱；**150ms** `ease-out`。
- **禁用**：背景 `#B0E0E6` 50% 透明度，文字 `#718096`，无悬停位移。

### 1. 职场白领类（互联网、教师、公职人员）

**风格**：简约干练、温和端庄；色调偏**沉稳蓝**（强调色与略深天蓝组合）。

| 按钮名称 | 适用场景 | 配色方案 | 尺寸与圆角 | 搭配图标 | 交互细节 |
|----------|----------|----------|------------|----------|----------|
| **职场穿搭生成** | 首页主按钮（选场景后触发推荐） | 背景 `#4682B4`，文字 `#FFFFFF`，无边框或边框 `#4682B4` | 高 48px，圆角 **16px** | **西装外套 + 领带**简线图标（或公文包与衣架组合） | Hover：背景 `#5a92c4`；Click：轻微压下；加载中按钮内显示**小衣架旋转**占位 |
| **通勤搭配推荐** | 搭配推荐页 `/outfit` 顶部主操作 | 背景 `#4682B4`，文字白；次态outline 版：底 `#E0F7FF`，字与边框 `#4682B4` | 主 48px / 16px；次 40px / **12px** | **乐福鞋 + 长裤**剪影（通勤意象） | Hover：outline 版背景略加深至 `#d0effa`；焦点环 `2px solid #87CEEB` |
| **解析结果确认** | 视频解析页结果区主按钮 | 背景 `#4682B4`，文字白；若与「全选」并列，次按钮用浅底钢蓝字 | 高 48px，圆角 **16px** | **勾选衣架**（衣架上挂一件单品 + 对勾） | Click 后短暂变为「已入库」态：底 `#B0E0E6`，字 `#4682B4`，**1.2s** 后恢复 |

**风格**：个性艺术、松弛随性；色调偏**明亮天蓝 / 低饱和蓝**，留白略多。

| 按钮名称 | 适用场景 | 配色方案 | 尺寸与圆角 | 搭配图标 | 交互细节 |

## 五、页面按钮适配说明

### 1. 首页 `/`

- **核心按钮基础样式**：场景卡片内 CTA 与底部「生成推荐」类操作为**主按钮**（48px、圆角 16px）；「自由搭配」为**次级**（浅天蓝底 + 钢蓝字、12px 圆角）。
- **随职业切换**：白领 → 主 CTA 采用 **#4682B4 沉稳实心**（职场穿搭生成）；创意 → **#87CEEB 明亮主色**（灵感穿搭生成）；学生 → **#87CEEB 或浅渐变**（校园穿搭生成）。图标按第四章表格替换，**文案**随职业微调（干练 / 灵感 / 活力）。
- **与穿搭主题适配**：场景卡片左侧统一为小**衣架/单品**占位图；选中卡片边框 **2px #87CEEB**，底纹 `#E0F7FF`。

### 2. 视频解析页 `/video`

- **核心按钮**：「解析」为主按钮（全宽或右对齐，**#4682B4**）；「保存选中到衣橱」与「解析结果确认」同系，白领预设为钢蓝实心，学生/创意可将保存按钮改为**浅底 + 钢蓝字**以减轻压迫感。
- **随职业切换**：确认入库按钮图标在白领为**勾选衣架**；创意可加**细边框**装饰；学生用**衣橱新增**图标文案。
- **适配细节**：结果列表每行左侧**鞋/上衣小图标**表示品类；主操作区背景条用 `#E0F7FF`，与解析中 Loading 的**天蓝进度条**（主色）一致。

### 3. 衣橱页 `/wardrobe`

- **核心按钮**：筛选 Chip 为**12px 圆角**；「去自由搭配」「从相册补充」等入口为次级按钮（`#E0F7FF` / `#4682B4` 字）。
- **随职业切换**：Chip 选中态白领加深描边（**#4682B4**）；创意选中态用**主色填充 + 白字**；学生选中态**浅蓝填充 + 钢蓝字**，更轻快。
- **适配细节**：网格卡片圆角 **16px**，悬浮时**衣架形阴影**（可选 `border-color #87CEEB` 加深），强化「衣橱」隐喻。

### 4. 搭配推荐页 `/outfit`

- **核心按钮**：「换场景」为文本链或幽灵按钮（`#4682B4`）；「生成 / 刷新推荐」为主按钮，职业映射同首页；卡片内「查看详情」为 40px 次按钮。
- **随职业切换**：白领强调**通勤搭配推荐**与分数徽章（`#4682B4` 描边）；创意突出**灵感穿搭**与**收藏/风格保存**；学生突出**日常搭配推荐**与轻松文案。
- **适配细节**：`OutfitCard` 背景白或 `#E0F7FF`，单品缩略图下加**小鞋/上衣分类角标**，与全站穿搭叙事一致。

### 5. 个人中心 `/profile`（或首启职业浮层）

- **核心按钮**：「保存职业与偏好」为主按钮（48px、16px 圆角），默认 `#4682B4`，与全站主操作一致；分组单选（职业大类）用**卡片式选项**，选中边框 `#87CEEB`、底 `#E0F7FF`。
- **随职业切换**：本页为**切换源**——保存后全站按第四章映射更新 Pinia / 本地存储；预览区展示**三类按钮小样**（缩略）便于用户确认视觉气质。
- **适配细节**：顶部可用**柔和天蓝渐变头图** + **衣架线稿**；无登录时说明文案置底灰色小字，主视觉仍保持穿搭与温柔蓝系。
