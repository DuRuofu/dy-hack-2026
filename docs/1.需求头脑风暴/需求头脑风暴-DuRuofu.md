# 搭子 — 需求头脑风暴

> 黑客松当天快速对齐：这个 App 到底要做哪些事？

---

## 用户视角：我打开"搭子"能做什么？

### 1. 建衣橱 — 我有哪些衣服？

| 来源 | 用户动作 | 系统做的事 |
|------|----------|-----------|
| 拍照上传 | 对着衣柜拍一张 | AI 自动裁剪、识别品类/颜色/风格，归类入库 |
| 抖音视频 | 看到博主穿搭，转发/粘贴链接 | 从视频关键帧中提取服装单品，加入"灵感衣橱" |
| 淘宝同步 | 授权淘宝账号 | 读取购买记录和收藏，自动生成衣物列表 |
| 手动录入 | 上传图片 + 填写标签 | 存入个人衣橱，可编辑属性 |

### 2. 选场景 — 我今天要去哪？

用户选择/输入场景，例如：
- 通勤上班、面试
- 约会、聚会
- 运动、旅行
- 雨天、降温、高温

### 3. 看推荐 — 今天穿什么？

- **一键搭配**：AI 从衣橱中挑出一套完整搭配（上衣+下装+鞋+配饰）
- **多套备选**：给出 2-3 套方案，附带搭配理由
- **缺件提示**：推荐方案中如果衣橱缺少某件单品，提示"你缺一双白鞋"

### 4. 逛灵感 — 别人怎么穿？

- 从抖音穿搭博主视频中浏览提取的穿搭方案
- 点击某套搭配，查看"类似单品我有哪几件"
- 一键复刻：用自己衣橱中的相似单品替换博主同款

### 5. 管衣橱 — 衣服越攒越多怎么办？

- 按品类/颜色/季节/风格筛选
- 标记常穿 / 很久没穿 / 想淘汰
- 穿搭记录：记录每天穿了什么，生成"本月穿搭回顾"

## 开发者视角：这个 App 需要哪些模块？

### 前端模块

| 模块 | 页面/组件 | 说明 |
|------|----------|------|
| 首页 | 场景选择 + 今日推荐 | 核心入口，选场景 → 看搭配 |
| 衣橱 | 衣物列表 + 筛选 | 网格/列表展示，支持分类筛选 |
| 拍照上传 | 摄像头/相册 + 预览 | 拍照后预览确认，提交识别 |
| 搭配详情 | 搭配卡片 + 单品详情 | 展示一套完整搭配，可点击查看单品 |
| 灵感页 | 抖音视频穿搭流 | 从视频提取的穿搭方案瀑布流 |
| 个人中心 | 设置、淘宝授权、风格偏好 | 用户配置 |

### 后端模块

| 模块 | 接口 | 说明 |
|------|------|------|
| 图像识别 | POST /api/recognize | 调用多模态 AI，识别图片中的服装 |
| 衣橱管理 | CRUD /api/wardrobe | 增删改查用户衣物 |
| 搭配推荐 | POST /api/outfit/recommend | 输入场景+衣橱，返回搭配方案 |
| 视频提取 | POST /api/video/extract | 从抖音视频中提取穿搭单品 |
| 淘宝同步 | GET /api/taobao/sync | 读取淘宝购买记录 |
| 用户管理 | /api/user | 注册登录、偏好设置 |

### AI 能力（复用 references 中的预研模块）

| 能力 | 对应 Demo | 用途 |
|------|----------|------|
| 图片多模态理解 | demo2 | 识别衣物品类、颜色、风格 |
| 视频链接解析 | demo1 | 从抖音视频提取关键帧 |
| 创意内容生成 | demo4 | 生成搭配文案/推荐理由 |
| 视频内容摘要 | demo3 | 提取视频中的穿搭信息 |
