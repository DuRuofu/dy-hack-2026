# 功能-接口映射

> 前端每个页面需要后端提供什么接口，一一对应

## 页面 → 接口 对照表

| 页面 | 用户操作 | 需要的接口 | 请求方式 | 返回数据 |
|------|---------|-----------|---------|---------|
| 首页 | 选择场景 | /api/scenes | GET | 场景列表 |
| 拍照页 | 上传图片识别 | /api/recognize | POST | 衣物信息 |
| 衣橱页 | 查看衣物列表 | /api/wardrobe | GET | 衣物数组 |
| 推荐页 | 获取搭配 | /api/outfit/recommend | POST | 搭配方案 |
| 灵感页 | 浏览视频穿搭 | /api/inspiration | GET | 穿搭列表 |

## 数据流

```
拍照上传 → AI识别 → 写入衣橱 → 选场景 → 推荐搭配 → 展示详情
```
